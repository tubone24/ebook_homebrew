Deploy to Heroku
================

Deploy for CLI
--------------

.. note::

    Before push heroku button, create your Heroku account.

Install Heroku for CLI ::

    $ heroku create <app name>
    $ heroku stack:set container
    $ git push heroku master
    $ heroku open
